Bases de control de movimiento

Descripción

El objetivo del curso es que el participante conozca e identifique las características de los bloques de función de control de movimiento, para poder utilizarlos en diferentes aplicaciones que conlleven el uso del control de movimiento básico.
Al participante se le entregará una guía teórica del curso.

Temario
  • Conceptos básicos para control de movimiento.
  • Tipos de movimiento
  • Bloques de función para control de movimiento
  • Comandos y parámetros
  • Secuencias de movimiento

Mini robótica industrial

Descripción

El objetivo del curso es que el participante conozca e identifique algunos tipos de robots utilizados en la industria, así como sus diferentes aplicaciones y consideraciones a tomar en cuenta, según el tipo de aplicación. Al participante se le entregará una guía teórica del curso.

Temario
  • Bases de robótica
  • Tipos de robots
  • Robot Gantry
  • Robot Delta
  • Robot articulado

Programación básica de PLC's

Descripción

El objetivo del curso es que el participante comprenda y aplique los diferentes elementos de la lógica de escalera a programas con aplicaciones reales, y con esto, tener las bases de programación en escalera para poder aplicar soluciones en cualquier PLC que se programe en este lenguaje.
Al participante se le entregará una guía teórica del curso.

Temario
  • Fundamentos teóricos del PLC
  • Programación en escalera
  • Entradas y salidas
  • Temporizadores
  • Contadores
  • Comparadores
  • Operaciones matemáticas
  • Subrutinas
  • Latch-Unlatch
